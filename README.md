**Smart Downloads Organizer** — это программа, которая автоматически наводит идеальный порядок в вашей папке «Загрузки». Она работает на заднем плане (в фоне), замечает появление новых файлов и сразу раскладывает их по правильным папкам (`Изображения`, `Документы`, `Архивы` и т.д.), а также следит за тем, чтобы старый мусор автоматически отправлялся в Корзину.

---

## ✨ Что умеет программа

- 📂 **Авто-сортировка:** Раскладывает новые файлы по папкам (`Images`, `Documents`, `Archives`, `Applications`, `Audio_Video`) в зависимости от их типа.
- 🧹 **Генеральная уборка:** При каждом запуске скрипт сначала разберет новые файлы, которые скопились в корне Загрузок.
- 🗑️ **Умная очистка (TTL):** Автоматически находит файлы в подпапках, которые лежат там дольше указанного вами срока (например, 30 дней), и безопасно отправляет их в **системную Корзину** (файлы всегда можно восстановить).
- ⚙️ **Умное определение путей:** Скрипт сам находит папку «Загрузки» на любом компьютере. Он отлично понимает как англоязычные системы (`Downloads`), так и русскоязычные локали Linux/Windows (`Загрузки`).
- 🛡️ **Защита файлов:** Если вы скачаете файл с именем, которое уже есть в целевой папке, программа не перезапишет старый файл, а добавит к новому цифры (временную метку).
- 📝 **История действий:** Все перемещения, удаления и ошибки аккуратно записываются в текстовый файл `organizer.log`.

## 🛠️ Как установить и запустить (Инструкция для всех ОС)

Перед началом убедитесь, что на вашем компьютере установлен **Python** (версии 3.8 или новее). Если его нет, скачайте его с [официального сайта python.org](https://python.org). При установке на Windows обязательно поставьте галочку **"Add Python to PATH"**.

### Шаг 1. Скачивание программы

Вы можете получить проект двумя способами:

* **Вариант А (Для разработчиков):** Склонируйте репозиторий через Git, выполнив команду в терминале:
  ```bash
  git clone https://github.com/ivanigonin13-ops/smart-downloads-organizer.git
  cd smart-downloads-organizer
  ```

* **Вариант Б (Для обычных пользователей):** Нажмите зеленую кнопку **Code** вверху этой страницы на GitHub, выберите **Download ZIP** и распакуйте скачанный архив в любую удобную папку (например, на Рабочий стол). Затем откройте терминал и перейдите в эту папку с помощью команды `cd`.

---

### Шаг 2. Установка и Запуск

Выберите инструкцию для вашей операционной системы:

#### 🪟 Инструкция для Windows
1. Откройте папку с распакованной программой.
2. Кликните по пустой строке с адресом папки вверху проводника, введите три буквы `cmd` и нажмите **Enter**. Откроется черное окно (Командная строка).
3. Установите необходимые компоненты, введя команду и нажав Enter:
   ```bash
   pip install -r requirements.txt
   ```
4. Запустите программу:
   ```bash
   python organizer.py
   ```
5. Программа начнет работу! Чтобы выключить её, просто закройте черное окно или нажмите `Ctrl + C`.

#### 🍏 Инструкция для macOS
1. Откройте программу **Терминал** (через поиск Spotlight или в папке Программы -> Утилиты).
2. Наберите слово `cd ` (с пробелом на конце), перетащите папку с программой из Finder прямо в окно Терминала и нажмите **Enter**.
3. Установите необходимые компоненты:
   ```bash
   pip3 install -r requirements.txt
   ```
4. Запустите программу:
   ```bash
   python3 organizer.py
   ```
5. Чтобы остановить программу, нажмите комбинацию клавиш `Control + C` в Терминале.

#### 🐧 Инструкция для Linux
1. Откройте терминал и перейдите в папку с проектом:
   ```bash
   cd /путь_к_папке/smart-downloads-organizer
   ```
2. Установите зависимости:
   ```bash
   pip3 install -r requirements.txt
   ```
3. Запустите скрипт:
   ```bash
   python3 organizer.py
   ```
4. Для остановки трекера нажмите `Ctrl + C`.

## 🥷 Как сделать так, чтобы программа работала сама (Автозапуск)

Чтобы не запускать скрипт вручную каждый раз, настройте автозапуск:

### 🪟 На Windows (В фоновом режиме)
1. Переименуйте файл `organizer.py` в `organizer.pyw` (буква **w** на конце означает, что программа будет работать невидимо, без черного окна).
2. Нажмите комбинацию клавиш `Win + R` на клавиатуре, введите команду `shell:startup` и нажмите **Enter**. Откроется системная папка автозапуска.
3. Нажмите правой кнопкой мыши по вашему файлу `organizer.pyw` -> **Показать дополнительные параметры** -> **Создать ярлык**.
4. Перенесите созданный ярлык в открывшуюся папку автозапуска. Готово! Теперь программа включается сама вместе с компьютером.

### 🍏 На macOS / 🐧 Linux (Фоновый запуск)
Если вы хотите запустить программу так, чтобы она работала в фоновом режиме даже после закрытия Терминала, используйте команду:
```bash
nohup python3 organizer.py &
```
Программа свернется в фон. Чтобы полностью её выключить, введите в Терминале команду `pkill -f organizer.py`.

## ⚙️ Как настроить программу под себя

В папке с программой есть файл `config.json`. Откройте его через любой текстовый редактор или Блокнот:

```json
{
  "watch_directory": "",
  "delete_old_files": true,
  "max_file_age_days": 30,
  "rules": {
    "Images": [".jpg", ".jpeg", ".png", ".gif", ".webp", ".svg"],
    "Documents": [".pdf", ".docx", ".doc", ".xlsx", ".txt", ".pptx", ".csv"]
  }
}
```

* **`watch_directory`**: По умолчанию здесь стоит `""` (пусто), что означает автоматический поиск стандартной папки Загрузок. Если вы хотите отслеживать другую папку, впишите туда полный путь, например: `"C:/Users/Имя/Desktop/МояПапка"`.
* **`delete_old_files`**: Поставьте `true`, если хотите, чтобы программа автоматически удаляла старые файлы из подпапок, или `false`, чтобы отключить эту функцию.
* **`max_file_age_days`**: Количество дней, по истечении которых файл считается старым и отправляется в Корзину.
* **`rules`**: Здесь прописаны категории и типы файлов. Вы можете добавлять свои расширения. Например, если вы хотите, чтобы файлы Photoshop (`.psd`) попадали в папку `Images`, просто допишите `".psd"` в список через запятую.
